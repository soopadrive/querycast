QueryCast

A user-controlled YouTube subscription feed. Transparent curation, no algorithm.

What is it?

QueryCast is an installable Progressive Web App that replaces YouTube's algorithmic home feed with a curation system you fully control. Every video that reaches your feed has passed your filters; every score that ranks them is a transparent linear combination of weighted signals you set.

It is not a YouTube replacement. You still watch videos via YouTube's player. QueryCast replaces discovery — the part where the algorithm decides what to surface.

Status

v0.1 — Stage 1 placeholder. Sign-in and the actual feed land in subsequent stages. The plan is at ~/plans/querycast.html.

Test mode only. Per ADR-002, this app is in Google's OAuth Testing tier. It cannot be used by anyone outside the manually-invited test list (capped at 100 users by Google policy).

Install

Desktop (Chrome, Edge, Brave)

  1. Visit the deployed URL.
  2. Click the install icon in the address bar (looks like a small monitor with an arrow).
  3. Confirm. QueryCast installs as a standalone app.

Android (Chrome)

  1. Visit the deployed URL.
  2. Tap the menu (three dots) → "Install app" or "Add to Home screen."

iOS (Safari)

  1. Visit the deployed URL.
  2. Tap the Share button (square with arrow).
  3. Scroll to "Add to Home Screen" → Add.

Run locally (developers)

cd /path/to/querycast
vercel dev    # serves at http://localhost:3000

Static files + any api/*.js functions are served. Service worker registers on first load.

Privacy

See Privacy Policy for what data the app accesses and where it lives. TL;DR: read-only YouTube scope + a Drive folder only QueryCast can read; nothing on our servers; no telemetry.
